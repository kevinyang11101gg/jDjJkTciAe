# Flask图书管理系统

## 核心功能
- 分类管理模块
- 图书管理模块
- 借阅管理模块
- 读者管理模块
- 账号管理模块
- 角色和权限管理模块
- 大模型智能聊天
- 个人中心模块
- 首页展示模块

## 适用场景
- 图书馆
- 学校图书馆
- 公共图书馆
- 企业图书馆

## 技术特点
- 基于Python和Flask框架，使用MySQL数据库
- 提供角色权限管理，确保数据安全
- 内置大模型智能聊天功能，方便用户互动
- 界面简洁易用，操作便捷
